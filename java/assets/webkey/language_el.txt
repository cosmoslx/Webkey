Low resolution -> Χαμηλή ανάλυση
It makes resolution half both in horizontal and vertical direction. -> Μειώνει την ανάλυση κατά το ήμισυ.  
JPEG is a lossy compression. It is fast and has a very good compression ratio, but the image gets noisy. PNG is much slower, the image size is larger, but it uses lossless compression. -> Με χρήση JPEG έχετε καλύτερη απόκριση αλλά χειρότερη εικόνα . Με χρήση PNG καλύτερη εικόνα αλλά πιο αργή απόκριση. 
Usually Android uses double buffering: while the phone uses one of the two framebuffers to display, it prepares the new image on the other one. Then swaps them. That's an old trick to avoid flickering. On some older phones (for example on HTC G1 aka Dream) may display some parts from the first, and some parts from the second framebuffer, but in this case usually the second framebuffer doesn't contains a whole image. This option forces Webkey to use only the first framebuffer. -> Χρησιμοποιήστε αυτήν την λειτουργία εάν η εικόνα φαίνεται αλλοιωμένη.  
Record images -> Εγγραφή εικόνων 
The phone starts to save images to a temporary folder /sdcard/webkey_TEMP/ as fast as it can. It uses the selected options (Low resolution, Horizontal, Flip, Png/Jpeg, Use only first framebuffer). After pressing Stop recording, the phone makes a zip archive from the images and sends it to the browser. -> Η συσκευή αρχίζει και καταγράφει εικόνες σε έναν προσωρινό φάκελο στην κάρτα SD (/sdcard/webkey_TEMP/). Αφού πατήσετε λήξη εγγραφής , η συσκευή συμπιέζει τις εικόνες σε ένα αρχείο Zip και τις στέλνει πίσω στον πλοηγό σας . Το αποτέλεσμα εξαρτάται από τις δυνατότητες της εκάστοτε συσκευής .

SMS mode -> Λειτουργία SMS
If you try to send special characters, like ő, ű in an SMS message, then these characters force Android to switch to unicode format, where the number of characters in a message drops from 160 to 70 in GSM networks. If SMS mode is checked, then lines with "works in SMS mode" have priority. -> Αν θέλετε να στείλετε ειδικούς χαρακτήρες όπως ő, ű και ελληνικά σε SMS τότε το όριο των χαρακτήρων θα πέσει από τους 160 στους 70. Εάν η λειτουργία SMS είναι ενεργή τότε οι σειρές με την ένδειξη “ Λειτουργεί σε SMS” έχουν προτεραιότητα .
